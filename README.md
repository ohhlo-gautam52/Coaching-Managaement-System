#  Coaching Management System (CMS)

A simple and efficient Coaching Management System built to manage students, teachers, attendance, and fees all in one place.

This project is designed to reduce manual work and help coaching institutes handle daily operations digitally.

---

##  Features

*  Student Management
*  Teacher Management
*  Attendance Tracking
*  Fees Management
*  Dashboard Overview
*  JSON-based data storage (no database setup required)

---

##  Tech Stack

* Node.js
* Express.js
* HTML, CSS, JavaScript
* JSON (as local database)

---

##  Project Structure

```
CMS-COMPANY-LEVEL
│
├── views/           # HTML pages
├── public/          # CSS & JS files
├── data/            # JSON data storage
├── routes/          # API routes
├── server.js        # Main backend server
├── package.json     # Dependencies
```

---

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/cms-project.git
cd cms-project
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Start the server

```
node server.js
```

or (if using nodemon):

```
npx nodemon server.js
```

---

##  Open in Browser

After running the server, open:

```
http://localhost:3000
```

(or check your port in server.js)

---

##  Important Notes

* This project uses `data.json` as a local database
* No external database (MySQL/MongoDB) is required
* Perfect for demo and learning purposes
